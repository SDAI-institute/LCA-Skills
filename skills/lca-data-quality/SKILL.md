Assess and communicate data quality for each significant inventory input and output.

Pedigree matrix dimensions (ecoinvent approach):
- **Reliability** — measured/verified vs. estimated/unknown
- **Completeness** — representative sample vs. single source
- **Temporal correlation** — data age relative to study reference year
- **Geographical correlation** — spatial match to study geography
- **Technological correlation** — match to actual technology in scope

Scoring guidance (1 = best, 5 = worst for each dimension):
- Score 1–2: reliable, complete, recent, geographically matched, technology matched
- Score 3: acceptable with documented limitations
- Score 4–5: estimated, outdated, or proxy data — flag for sensitivity analysis

Data gap strategy:
1. Prefer measured or published LCI datasets (ecoinvent, GaBi, ELCD) over estimates.
2. Use proxy processes when exact matches are unavailable; document the substitution.
3. Apply system expansion when allocation cannot be avoided and co-products exist.
4. Flag high-uncertainty exchanges that contribute >5% to any characterization result.

Agent guidance:
- In inventory work, attach a brief quality note to each exchange: source, year, geography.
- Prioritize data quality improvements for exchanges identified as high contributors in LCIA.
- In interpretation, recommend sensitivity runs for the three highest-uncertainty dominant exchanges.
